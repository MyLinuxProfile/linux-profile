# Resume

Profile file management.

## Usage

```bash
linuxp profile --help

=============== Output ===============

usage: linuxp profile [-h] [--url URL] [--switch SWITCH] [--output OUTPUT]

options:
  -h, --help       show this help message and exit

Usage: linuxp profile [OPTIONS]:
  --url URL        URL to download and sync profile... Ex: --url https://linuxprofile.com/linux_profile.json
  --output OUTPUT  File name to save.................. Ex: --output 'my_linux'
  --switch SWITCH  File name for profile switching.... Ex: --switch 'linux_profile.json'

=============== Output ===============
```
